#### Install AND Run

- you need to install poetry
    
```sh
      - pip install poetry
      - poetry install --no-root 
      ## or you can just run this
      - poetry install
```  
- to run linear regression
```sh
    - poetry run linear
```
- to run logistic regression
```sh
    - poetry run logistic
```

- to run Decision tree
```sh
    - poetry run dtree
```

- to run SVM
```sh
    - poetry run svm
```

- to run k-means
```sh
    - poetry run kmean
```
